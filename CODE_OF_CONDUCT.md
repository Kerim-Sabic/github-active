# Code Of Conduct

GitHub Active is an open-source project for transparent developer tooling.

Expected behavior:

- Be respectful and specific.
- Discuss technical tradeoffs directly.
- Prefer reproducible examples over vague claims.
- Do not use the project to coordinate spam, fake engagement, or deceptive profile activity.

Unacceptable behavior:

- Harassment or personal attacks.
- Posting secrets, tokens, or private user data.
- Requests to bypass platform rules or misrepresent contribution history.

Maintainers may remove comments, close issues, or block participation that violates these expectations.
